Conteúdo da Semana 1

Bootstrap é um framework de código aberto que facilita a criação de sites responsivos e visualmente atraentes. Ele fornece um conjunto de classes e componentes predefinidos.

Grid System: O Bootstrap utiliza um sistema de grid de 12 colunas que permite o layout flexível de páginas em diferentes dispositivos. As classes, como .container e .row, ajudam a estruturar o conteúdo.

Componentes Pré-Definidos: Bootstrap oferece uma variedade de componentes prontos para uso, como navegação, botões, formulários, carrosséis e muito mais.

Responsividade: Bootstrap é conhecido por sua capacidade de criar sites que se adaptam a diferentes tamanhos de tela, tornando-os móveis e amigáveis para dispositivos.
